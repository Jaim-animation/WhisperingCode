1. Pixels


A computer screen consists of tiny squares called pixels. Each pixel contains three subpixels: one red, one blue, and one green. By controlling the brightness of these subpixels, a pixel can display different colors. A single pixel can show 256 * 256 * 256 = 16,777,216 colors. This is how your screen displays images.


2. Binaries


To control the screen, computers use thousands of tiny machines called microchips, which are made using transistors that can be as small as a few nanometers.

These microchips use electrical circuits that can be turned on or off. A state of on is represented by 1, and a state of off is represented by 0. This binary system, also known as machine code, is the foundation of computer operations. You are reading this article because some circuits are displaying it on your screen.

However, not all computers use the same machine code. Just as different brands of lights work differently, different computer architectures use different machine codes. Some popular architectures are:

  1. x86
  2. x64
  3. ARM
